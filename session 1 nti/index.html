7.		 Course Learning Outcomes (CLO)

After the completion of this course, a successful student will be able to do the following:

  1. Develop simple algorithms and flow charts to solve a problem using top down design principles.
  2. Write efficient and well-structured computer algorithms/programs.
  3. Formulate iterative solutions and array processing algorithms.
  4. Use recursive techniques, pointers and searching methods.
  5. Understand fundamental data structures and their implementation.
  6. Know complexity of operations like insert, delete, search on data structures.
  7. Choose a suitable data structure for applications.
  8. Design programs using data structures (hash tables, trees, heaps, graphs, etc.).
  9. Assess efficiency tradeoffs among data structure implementations.
  10. Implement and apply algorithms effectively.
